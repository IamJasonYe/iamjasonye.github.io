笔记电子化

很久以来，也就是从初中以来，我的一贯记笔记方式都是用一个活页本去记录我日常的笔记，包括学校里上课的笔记，灵光一闪的idea。

久而久之，笔记变得越来越多，有时候记笔记时没有很好总结，所以内容比较碎片化，对于后续的管理和整理带来了巨大的麻烦。我已经不止一次因为把笔记夹在草稿纸之间，然后丢过一些重要的笔记，现在想想真是愚蠢呀。

再看近几年，各种平板电脑层出不穷，而且移动操作系统（如iPadOS）和多任务执行给很多学生和商务人士带来更加轻便化的效率改变。比如iPad Pro, 除了有官方宣传的能够用来编辑视频以外，基本上任何简易的处理，如写文档，回复邮件，都能够打开即用，十分方便。

结合多方需求，因而我果断在去年黑五的时候，买了个iPad Air，开始我的笔记电子化道路。而到现在已经两个月时间了，我觉得这个“投资”还是花得挺值的。

笔记电子化的一个很大要素就是手写笔记大部分时候会比打字快。解决这点的就是这些平板所相应的stylus,所谓的手写笔。因为在很多时候，我们如果用笔记本电脑去记笔记，可能会遇到一些打字太慢的情况，比如手抄公式或者是特殊符号，这会让人十分痛苦，我之前体验过一段时间，经常是为了打一个公式，就落下了一大段笔记没抄。

至于记笔记所使用的软件，市面上有很多。考虑到跨平台的原因，我就在可选的选项里选择了微软的OneNote，毕竟我之前也用了很久吧，对这个app还是挺满意的。

将来有时间，可以再来分享一下如何用OneNote系统地记笔记（挖个坑）。

开始Bullet Journal

在翻阅各种iPad使用技巧和分享的时候，就看到了不少关于“子弹笔记”(英文缩写是BuJo)的介绍，也就很好奇别人是怎么做的。

看了几个YouTube 上的视频, 感觉就是大家所说的“手帐”，用来记录生活和计划行程。很明显，BuJo好处有很多，就是能够直观看到每天的安排，然后相应的完成进度，对于自己没做到的东西还能写上原因，便于日后反省，同时每隔一段时间还能从各个板块里面获得直观的总结，这样一套东西下来，基本上把很多现有的app给涵盖进去了，类似自创了一个更加密集的管理工具。

为了更加系统地了解BuJo, 我就直接去了原作者Ryder Carroll的网站去了解。总结来说，BuJo主要包含了几个部分，一个是笔记系统，一个是笔记内容，还有笔记模板。这三者分别解决了几个问题：

  • 如何快速记笔记？ -> 将内容符号化，关键词化 -> 建立自己的符号语言系统
  • 日常记录有什么？ -> 笔记内容
  • 怎么有效管理笔记？ -> 建立自己模板，建立索引，建立tracker，及时迁移
  • 如果提升执行效率 -> 构建工作良性循环(计划 => 执行 => 调整 => 计划 => ...)

我打算简直一段时间（至少一个月时间），到时候再来总结一些东西，看看是否真的适合自己吧。

Moreover, the outliers, some bad ones of the keypoints will exacerbate the perspective transform matrix. The task becomes much more complicated.
